# 图标总览

这里按图标类型罗列当前内置的全部图标。点击任意图标卡片即可复制图标名。

- 普通图标：`mc-xxx`，支持改色。
- 按键映射图标：`mc-key-xxx`，保持原色。
- 彩色图标：`mc-x-xxx`，保持原色。
- MDI 像素图标：`mdi-xxx`，支持改色，基于 [Material Design Icons](https://pictogrammers.com/library/mdi/) 24×24 像素化渲染，使用方法见 [自定义 SVG 图标](/components/icon#自定义-svg-图标)。
